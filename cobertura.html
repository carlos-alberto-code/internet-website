Zonas Urbanas

Tecnología principal: Fibra Óptica

Características:

  • Velocidades simétricas hasta 200 Mbps
  • Instalación: $200
  • Máxima estabilidad y velocidad
  • Ideal para hogares y negocios

  • Ideal para comunidades rurales

Requisitos de Cobertura

Información importante sobre los requisitos técnicos para cada tecnología

Fibra Óptica - Requisitos

Ubicaciones elegibles:

  • Centros urbanos principales
  • Zonas con infraestructura de postes
  • Distancia máxima: 100m desde poste
  • Acceso para cableado aéreo

Ventajas:

  • Mayor velocidad y estabilidad
  • No depende del clima
  • Velocidades simétricas
  • Menor costo de instalación
Inalámbrico - Requisitos

Ubicaciones elegibles:

  • Línea de vista hacia torres
  • Zonas rurales y semi-urbanas
  • Comunidades alejadas
  • Espacios para antena exterior

Ventajas:

  • Cobertura en zonas remotas
  • Instalación rápida
  • No requiere infraestructura previa
  • Ideal para comunidades rurales
